#Protótipo de Tradutor de Libras para Português
Este projeto faz parte do meu TCC em Engenharia de Software aprovado em dezembro de 2025 e apresenta um protótipo inicial que utiliza Python, MediaPipe, OpenCV e Scikit-learn para reconhecer e categorizar configurações de mãos da Libras em vídeo, servindo como prova de conceito para um tradutor de Libras para português.
Estes arquivos são referentes ao projeto inicial e ainda passarão por algumas alterações e melhorias.
